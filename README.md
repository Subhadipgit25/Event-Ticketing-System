# 🎟️ Stellar Soroban Event Ticketing System

## 📌 Project Description

This project is a decentralized Event Ticketing System built using Soroban smart contracts on the Stellar blockchain. It allows users to create events and purchase tickets securely without relying on centralized platforms.

## 🚀 What it does

- Event organizers can create events with a fixed number of tickets.
- Users can buy tickets directly from the blockchain.
- Ensures transparency — no fake tickets or overselling.
- Prevents double spending of tickets.

## ✨ Features

- 📅 Create Events with limited ticket supply  
- 🎟️ Buy Tickets securely using blockchain authentication  
- 🔒 Prevent overselling (ticket limit enforced in contract)  
- 📊 View event details (tickets sold, remaining tickets)  
- ⚡ Fast and low-cost transactions on Stellar
